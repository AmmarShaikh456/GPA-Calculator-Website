How GPA is Calculated

  • Each class is assigned a level: Onlevel (max 5.0), Honors (max 5.5), or AP (max 6.0).
  • Enter your grade (out of 100) for each class.
  • GPA for each class: max GPA for level − 0.1 × (100 − grade), floored at 0 for grades below 70.
  • Weighted GPA: Average of all class GPAs using their respective max values.
  • Unweighted GPA: Average of all class GPAs, always using 5.0 as the max (regardless of level).
